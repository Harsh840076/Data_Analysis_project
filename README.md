# Data_Analysis_project
# Supply Chain Delay Prediction & Data Analysis

## Project Overview
This project focuses on analyzing supply chain delivery data and predicting late delivery risk using Machine Learning. The dataset was cleaned, explored, visualized, and used to train predictive models.

The main objective is to help businesses identify shipment delays in advance and improve delivery performance.

---

## Problem Statement
Late deliveries can affect customer satisfaction and business operations. This project helps in predicting whether an order is likely to be delayed based on historical order and shipment data.

## Project Workflow

### 1. Data Cleaning
- Handled missing values
- Removed duplicates
- Converted data types
- Fixed inconsistent values

### 2. Exploratory Data Analysis (EDA)
- Delivery status analysis
- Shipping mode comparison
- Region-wise order trends
- Sales and profit analysis
- Delay percentage by category

### 3. Feature Engineering
- Created delay-related features
- Encoded categorical variables
- Selected important columns for training

### 4. Model Training
Used Machine Learning models such as:

- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier

### 5. Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## Key Insights
- Certain shipping modes had higher late delivery rates.
- Some regions showed more delays than others.
- Delivery status strongly impacts business performance.
